Предоставляемые материалы:
  - ./Задание.pdf - файл с вариантами задания.
  - ./Оформление отчета.docx - шаблон отчёта с указанимями по оформлению.
  - ./Пример_выполнения_на_симуляторе_JFLAP.pdf - презентация с описанием хода выполнения лабораторной в стимуляторе JFLab.
  - ./simulators - предлагаемые утилиты для выполнения практической части задания.
  - ./lab_test.txt - файл с входными значениями для функции от 4 переменных для проверки корректности реализованной машины Тьюринга, который может быть импортирован в симулятор JFLab.

Выбор варианта задания:
  - Вариант задания, выбираемый из предоставленного файла ./Задание.pdf, соответствует номеру студента в списке группы по модулю 10.
  - В файле ./Задание.pdf также представлены задания повышенной сложности. При его выборе необходимо согласовать это с преподавателем.

Отчет по заданию предоставляется в электронном формате:
  - отчет (docx);
  - результат выполнения задания (файл, экспортированный из утилиты).

Форма отчета стандартная: титульный лист; цель и задачи с указанием номера варианта и условия; ход работы, включающий таблицу или граф построенной программы для машины Тьюринга; выводы.
